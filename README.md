# Raidouz (source code repository)
A 8-bit Shoot-em up game made completely in Lua with Pico-8 virtual development environment.

Features:
- **Homemade particles managment** (shoots, explosions etc...)
- Enemy waves (enemies appear in timed waves)
- **2** weapon (you can switch between one and the other)
- Boss fights with **dynamic patterns** (bosses change their patterns the more you go through the fight)
